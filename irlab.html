Abstract

The aim of this project is to review and compare some of the promiment techniques that have been used for Twitter Sentiment Analysis(TSA) in recent literature. The case study particularly looks at three type of supervised TSA methods, namely, Lexicon based, Machine Learning based, and combination of both. The empirical study involves implementation of these techniques as well as comparsion of their performance on a well-known Twitter dataset. We also aim to provide the detailed results of this comparison and the underlying implemented functions for usage.
